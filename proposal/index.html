Universal Wearable Integration - Technical Details

Smart architecture using Terra API

What is Terra API?

Terra (https://tryterra.co) is a unified wearable API that connects to 20+ fitness devices and platforms. Instead of integrating each wearable separately (which would take 6-8 weeks), we integrate Terra once (2 weeks) and get access to ALL supported wearables.

How It Works:

  1. User Selects Wearable: In RISE app, user taps "Connect Wearable" ‚Üí chooses their device (Garmin, Fitbit, Apple Watch, etc.)
  2. Terra OAuth Flow: Terra handles authentication for that specific wearable
  3. Data Sync: Terra fetches data from the wearable and converts it to a unified format
  4. Webhook to RISE: Terra sends real-time updates to RISE backend when workouts complete
  5. RISE Processes: We receive standardized JSON regardless of which wearable user has
